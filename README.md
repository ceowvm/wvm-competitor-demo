# WVM Competitor Research — версия 2

Исправленная версия демо, которая не упирается в 60-секундный тайм-аут Netlify.

## Как работает

1. `/api/analyze/start` запускает OpenAI Responses API в `background: true` и сразу возвращает ID задачи.
2. Браузер проверяет `/api/analyze/status` раз в 3 секунды.
3. После завершения отображается таблица и становится доступна выгрузка CSV.

## Как заменить предыдущую версию

1. Распакуйте архив.
2. Откройте существующий GitHub-репозиторий.
3. Удалите старые файлы проекта.
4. Загрузите всё содержимое новой папки.
5. В Netlify откройте `Deploys` и дождитесь автоматического deploy.

Если deploy не запустился:

`Deploys → Trigger deploy → Clear cache and deploy site`

## Переменные Netlify

Обязательная:

`OPENAI_API_KEY = ваш ключ sk-proj-...`

Необязательная:

`OPENAI_MODEL = gpt-5.6-luna`

Если `OPENAI_MODEL` не задана, версия 2 сама использует `gpt-5.6-luna`.
