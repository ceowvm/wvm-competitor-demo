# WVM Competitor Research — демо

Рабочий MVP веб-сервиса, который принимает:

- нишу;
- регион;
- количество компаний (1–10);

и возвращает таблицу:

- название;
- официальный сайт;
- телефон.

Результат можно скачать в CSV.

## Что находится внутри

```text
public/
  index.html
  styles.css
  app.js
netlify/
  functions/
    analyze.mjs
netlify.toml
```

Фронтенд размещается как обычная страница, а `analyze.mjs` работает как защищённая серверная функция. API-ключ не попадает в браузер.

## Быстрый запуск на Netlify Free

### 1. Загрузите проект в GitHub

1. Распакуйте ZIP.
2. Создайте новый пустой репозиторий в GitHub.
3. Загрузите в него всё содержимое папки `wvm-competitor-demo`.

Важно: загружать нужно не саму внешнюю папку, а файлы и папки внутри неё.

### 2. Подключите репозиторий к Netlify

1. Откройте Netlify.
2. Выберите **Add new project** → **Import an existing project**.
3. Подключите GitHub и выберите репозиторий.
4. Netlify прочитает настройки из `netlify.toml`.
5. Нажмите **Deploy**.

### 3. Добавьте OpenAI API-ключ

В проекте Netlify откройте:

**Project configuration → Environment variables → Add a variable**

Добавьте:

```text
OPENAI_API_KEY = ваш_ключ
```

Дополнительно можно добавить:

```text
OPENAI_MODEL = gpt-5.6
```

После добавления переменных запустите новый Deploy.

## Важно про оплату

Размещение MVP на Netlify может работать на бесплатном плане.  
Запросы к OpenAI API оплачиваются отдельно от подписки ChatGPT Plus.

## Ограничения демо

- максимум 10 компаний за запрос;
- максимум 10 запусков в час с одного IP;
- серверная функция ожидает ответ не более 55 секунд;
- телефон берётся только с официального сайта;
- если телефон не подтверждён, выводится «Не найден»;
- результаты рекомендуется перепроверять перед использованием в рассылках и CRM.

## Почему не обычный Netlify Drop

Обычный drag-and-drop хорошо подходит для статических HTML/CSS/JS-страниц. В этом проекте есть серверная функция и секретный API-ключ, поэтому проект лучше подключить через GitHub или развернуть через Netlify CLI.

## Локальный запуск

При установленном Node.js:

```bash
npm install -g netlify-cli
netlify dev
```

Для локальной работы создайте в корне `.env`:

```text
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-5.6
```

Не загружайте `.env` и API-ключ в GitHub.
